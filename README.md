# PFP Admin - Административная панель

Веб-приложение для управления финансовыми продуктами и модельными портфелями пенсионного планировщика.

## 🚀 Технологии

- **React 18** - UI библиотека
- **TypeScript** - Типизация
- **Vite** - Сборщик
- **TailwindCSS** - Стилизация
- **Shadcn/UI** - UI компоненты
- **React Router** - Маршрутизация
- **Axios** - HTTP клиент

## 📦 Установка

```bash
npm install
```

## 🛠 Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## 🏗 Сборка

```bash
npm run build
```

Собранные файлы будут в папке `dist`

## 🚢 Деплой на Vercel

1. Подключите репозиторий к Vercel
2. Vercel автоматически определит настройки из `vercel.json`
3. Приложение будет развернуто автоматически

## 🔐 Авторизация

- Password: `admin123`

## 📡 API

Базовый URL бэкенда задаётся переменной **`VITE_API_BASE_URL`** (полный путь с `/api`).

- Локально: скопируй `.env.example` → `.env.local` и подставь свой URL
- **Vercel**: Project → Settings → Environment Variables → `VITE_API_BASE_URL`
- Если переменная не задана, используется прод: `https://pfpbackend-production.up.railway.app/api`

Документация API: `{origin без /api}/api-docs` (для дефолтного бэка — `https://pfpbackend-production.up.railway.app/api-docs`)

## 📱 Функциональность

### 1. Авторизация
- Страница входа с JWT токенами
- Автоматическое сохранение токена в localStorage
- Защита маршрутов

### 2. Управление продуктами
- Просмотр списка продуктов
- Создание и редактирование продуктов
- Управление таблицей доходностей
- Клонирование дефолтных продуктов

### 3. Управление портфелями
- Просмотр списка портфелей
- Создание и редактирование портфелей
- Настройка риск-профилей (Консервативный, Сбалансированный, Агрессивный)
- Управление инструментами в портфеле

### 4. Настройки системы
- Просмотр системных настроек
- Редактирование значений настроек
- Поддержка различных типов данных (строка, число, объект)

## 📁 Структура проекта

```
src/
├── components/      # React компоненты
│   ├── ui/        # UI компоненты (Shadcn/UI)
│   └── Layout.tsx # Основной layout
├── contexts/      # React контексты
│   └── AuthContext.tsx
├── lib/           # Утилиты и API
│   ├── api.ts     # API клиент
│   └── utils.ts   # Вспомогательные функции
├── pages/         # Страницы приложения
│   ├── Login.tsx
│   ├── Dashboard.tsx
│   ├── Products.tsx
│   ├── Portfolios.tsx
│   └── Settings.tsx
├── App.tsx        # Главный компонент
├── main.tsx       # Точка входа
└── index.css      # Глобальные стили
```

## 🔧 Конфигурация

Все настройки API находятся в `src/lib/api.ts`. Базовый URL можно изменить там же.
